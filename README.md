### 关于我

我现在主要在做前端开发，也在持续学习 AI 工程、Agent 工作流、知识库和项目文档化。

比起只把一个需求做完，我更在意一件事背后的链路：为什么要做、它在业务里解决什么问题、代码结构会不会变成长期负担、能不能沉淀成以后还可以复用的经验。

我的博客叫 **卷娄的小屋**。那里会写技术，也会写生活、实习、焦虑、表达和成长。它不是一个只展示结果的地方，更像是我把模糊想法慢慢讲清楚的过程记录。

### 最近在关注

- 用 `React`、`TypeScript`、`Next.js` 做更稳定、更好维护的前端工程。
- 在 AI 产品和浏览器插件里，把复杂表单、用户输入和自动化流程拆得更清楚。
- 学习 `Agent`、`Workflow`、`Context Engineering`，理解 AI 系统什么时候该简单，什么时候值得复杂。
- 把项目知识整理成结构、约束和验证，让 AI coding agent 能更可靠地参与工程。
